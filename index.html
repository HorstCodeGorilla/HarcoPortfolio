Ben een junior web developer in opleiding bij Code Gorilla. Ik ben 21 jaar oud ben geboren in Emmen. Heb totaal 7 jaar in de horeca gewerkt/stage. Daar naast heb ik mijn vader altijd mee geholpen bij zijn eigen bedrijf. Nu heb ik de kans gekregen om een studie bij Code Gorilla te volgen. Hier ben ik Code Gorilla ook erg dankbaar voor dat ik deze kans heb gekregen. Voor mij is deze opleiding zo super in elkaar gezet dat ik deze kans ook met 2 handen aan heb gepakt.
